# Task 2 - To-Do List Web App

## Objective
Create a dynamic To-Do list web application where users can add, remove, and mark tasks as complete using Vanilla JavaScript.

## Tools Used
- VS Code
- Chrome Browser
- Live Server

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Instant UI updates without page reload

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation, Event Handling)

## Learning Outcome
This project helped me understand:
- DOM Manipulation
- Event Handling
- Dynamic UI updates using Vanilla JavaScript
